  • git status muestra el estado de los archivos desde la ultima confirmacion si existiese el archivo
  • untracked el archivo no se le esta haciendo seguimiento
  • staged: el archivo ha sido modificado pero aun no estan confirmados
  • modified: el archivo difiere con respecto a lo almacenado en la ultima confirmacion
  • commited los cambios fueron almancenados en una confirmacion
  • git add < file agrega al INDEX el/los archivos seleccionados
  • git rm --catched quita INDEX el archivos seleccionados
  • git restore --staged < file remueve los cambios de los archivos agregados al INDEX
